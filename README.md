# 🤖 Telegram AI Agent (n8n + AI Automation)

A production-style AI agent built using **n8n and LLMs** that can execute real-world tasks directly from Telegram.

---

## 🚀 What This Does

This is not just a chatbot.

It is a **multi-agent automation system** that can:

- 📧 Send, read, and reply to emails  
- 📅 Create and manage calendar events  
- 👤 Store and retrieve contacts  
- 🌐 Search the web in real-time  
- 🐦 Generate and post content on X  
- 🔗 Execute multi-step workflows  

---

## ⚡ Example

User input:
> Find AI news → email summary → post on X

Agent automatically:
1. Searches latest AI news  
2. Generates summary  
3. Sends email  
4. Posts on X  

---

## 🏗️ Architecture

- Main Agent (Telegram interface)
- Sub-agents:
  - Email Agent  
  - Calendar Agent  
  - Contact Agent  
  - X Posting Agent  

Each agent is modular and connected via workflows.

---

## 🛠️ Tech Stack

- n8n (workflow automation)
- Telegram API
- LLM (Claude / AI models)
- Google APIs (Gmail, Calendar, Sheets)
